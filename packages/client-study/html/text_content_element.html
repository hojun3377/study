<blockquote>

컨텐츠 내의 텍스트가 인용문임을 나타냅니다.

contents

  • Flow content

attributes

  • global attribute
  • cite
    인용문의 출처 문서나 메시지를 가리키는 URL

example

The <blockquote> HTML element indicates that the enclosed text is an extended quotation. Usually, this is rendered visually by indentation (see Notes for how to change it). A URL for the source of the quotation may be given using the cite attribute, while a text representation of the source can be given using the <cite> element.

description list

설명 목록을 나타내는 리스트를 나타냅니다.
관련 태그로는 dl, dt, dd가 있습니다.

<dl>

설명 목록을 나타냅니다.
주로 용어사전 구현이나 메타데이터(키-값 쌍 목록)를 표시할 때 사용합니다.

contents

  • Optional:
    • <script>
    • <template>
  • Either:
    하나 이상의 <dt> 와 하나 이상의 <dd> 요소로 구성된 그룹 0개 이상
  • Or:
    하나 이상의 <div>

attributes

  • global attribute

<dt>

용어를 나타냅니다.
dl안에서만 사용이 가능하며, WHATWG HTML에서는 dl의 자손인 div안에서도 사용이 가능합니다.

contents

  • Flow content (header, footer, 구획 콘텐츠, 제목 콘텐츠는 불가능)

attributes

  • global attribute
